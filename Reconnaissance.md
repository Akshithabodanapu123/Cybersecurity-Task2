# Reconnaissance

## Introduction
Reconnaissance is the process of gathering information about a target before conducting security testing.

## Activities Performed

### WHOIS Lookup
Used WHOIS services to collect domain registration information.

### DNS Enumeration
Collected DNS records to understand domain infrastructure.

### Technology Detection
Identified technologies and services used by the target website.

## Learning Outcome
Learned how to gather publicly available information about a target using reconnaissance techniques.
