# Network Security & Scanning

## Project Overview

This project demonstrates basic network scanning and reconnaissance techniques using Python and Nmap.

## Objective

To identify open ports and analyze network services running on a target system.

## Tools Used

* Python
* Socket Programming
* Nmap
* Termux
* GitHub

## Python Port Scanner

The Python Port Scanner scans ports from 1 to 100 and identifies open ports using the socket module.

### Features

* Scans target IP address or domain name
* Detects open ports
* Uses TCP connections
* Simple command-line interface

## Sample Output

Port 22 is OPEN

Port 80 is OPEN

Scan Completed!

## Nmap Scan

Nmap was used to verify the open ports discovered by the Python scanner.

## Findings

| Port | Service | Status |
| ---- | ------- | ------ |
| 22   | SSH     | Open   |
| 80   | HTTP    | Open   |
